# My personal Website!
I originally had a project of my website, called personal-website. However, I realized that I was overcomplicating everything and decided to start from scratch with something more simple and in a framework I am more familiar with.

What is currently planned is that the frontend be Vue 3 and the backend be Django. The reason I decided to move away from Express.js is because Django comes with a lot of features that I originally planned to implement myself, and I recognized that it would've taken me too long to finish the project. I plan on publishing the first version of the website before New Year's Eve, December 31 2021.

If you have somehow come across this repo, and have a recommendation, feel free to open an issue with the tip!
