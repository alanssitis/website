Here are some of the projects I have worked on. Feel free to reach out if you want to collaborate on something!

- [**in-toto Contributor**](https://github.com/in-toto/): Contributor to the in-toto open source project.
- [**Puck-It: Smart Air Hockey Table**](https://www.puck-it.com/): Won first place in the Spark challenge for the best senior design project in ECE.
- [**Autonomous Kart Software Stack**](https://github.com/Autonomous-Motorsports-Purdue/AMP_ASSV3/): Built v3 of AMP's (Autonomous Motorsports Purdue) Autonomous Software Stack on ROS 2 Foxy.
- [**SSL Stack Detection**](https://github.com/alanssitis/ssl-stack-detection): Preliminary work to see whether you can use ciphersuite performance as a side channel to determine and classify the client's SSL stack.
- [**Purdue eCTF Efforts**](https://ectf.mitre.org/wp-content/uploads/2023/04/2023_eCTF_Purdue_Poster.pdf): Ported the lightweight crypto library Ascon to the target device (TI TM4C123GH6PM) and implemented the PRNG. Was not very good, but was a learning experience.
- [**Security Games**](https://github.com/alanssitis/security-games): The course project for the Probabilistic Methods course is a summary of interdependent security games. I wanted to do a project covering the intersection of the two classes I was taking at the time, Game Theory and Probability.
- [**Micro C Compiler**](https://github.com/alanssitis/course-compiler-project): Compiler written in Rust written for the compilers course ECE 573.
- [**alanchungma.com**](https://github.com/alanssitis/website): Astro static website hosted on Vercel. (This site!)
- [**IoT Device Hacking**](https://github.com/TSELab/IoT-Device-Hacking): Briefly messed around with IoT device hacking.
- [**Purdue Rekor Website**](https://github.com/TSELab/rekor-website): Worked on Purdue's rekor website as a VIP project in Spring 2022.
